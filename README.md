# AI Assignment - Name Matching & Recipe Chatbot

## Features
- Task 1: Name matching with FuzzyWuzzy and FAISS
- Task 2: Recipe suggestions based on ingredients

## Installation
```bash
pip install -r requirements.txt
```

## Run Locally
```bash
streamlit run app.py
```

## Usage
1. Navigate to Name Matcher page
2. Enter a name to find similar matches
3. View fuzzy and FAISS results

For recipes:
1. Go to Recipe Bot page
2. Enter ingredients (comma-separated)
3. Get matching recipes with instructions
